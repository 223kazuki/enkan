# enkan

Enkan(円環) is a microframework implemented Middleware pattern like ring or connect.  

## Concept

- Minimal
- Ease of development
- Ease of operation

### Minimal

- Middleware pattern

### Ease of development

- Faster startup (Preventing to scan classes)
- Reloading classes
- Trace execution of middlewares
- Alert misconfiguration
